# go-antizapret

## Description

Golang module to work with Antizapret PAC file: to detect proxy for blocked URL.

## Usage

See example in [cmd/example/main.go](cmd/example/main.go).

`import "github.com/antonsoroko/go-antizapret"`

## Credits
